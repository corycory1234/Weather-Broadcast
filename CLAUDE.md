## 技術棧
- 前端: React 19 + TypeScript
- CSS: Tailwind

## 專案架構
| 目錄 | 用途 |
|------|------|
| `src/locales/zh-TW` | 繁中i18n |
| `src/locales/en` | 英文i18n |
| `src/types` | TypeScript型別 |
| `src/pages` | 各路由主要page |
| `src/components` | React元件 |
| `src/router/routeIndex.tsx` | 路由設定 |
| `src/services/axiosInstance.ts` | axios網路請求實例化 |
| `src/index.css` | Tailwind CSS主要設定 |
| `src/main.tsx` | 前端主要main.tsx父層設定 |
| `docs/` | 文檔 |
